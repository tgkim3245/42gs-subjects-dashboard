# 42 Subject Dashboard Extension

42cursus 본과정 카뎃들의 Circle 00부터 Circle 06까지의 주요 과제 진행 현황을 편리하게 시각화하고 정렬/필터링할 수 있는 크롬 익스텐션 대시보드입니다.

다른 컴퓨터에서 이 익스텐션을 설치하고 실행하는 방법을 아래 단계별로 안내해 드립니다.

---

## 1단계: 익스텐션 소스코드 다운로드 (다른 컴퓨터에서)

다른 컴퓨터로 소스코드를 복사하거나 다운로드해야 합니다. 두 가지 방법 중 편한 방법을 선택하세요.

### 방법 A: Git으로 클론하기 (추천)
컴퓨터에 Git이 설치되어 있다면, 터미널(또는 CMD)을 열고 아래 명령어를 입력하여 프로젝트를 가져옵니다.
```bash
git clone https://github.com/tgkim3245/42gs-subjects-dashboard.git
```

### 방법 B: ZIP 파일로 다운로드하기
1. 웹 브라우저로 [저장소 주소(https://github.com/tgkim3245/42gs-subjects-dashboard)](https://github.com/tgkim3245/42gs-subjects-dashboard)에 접속합니다.
2. 우측 상단의 **`Code`** 초록색 버튼을 클릭한 후, **`Download ZIP`**을 눌러 다운로드합니다.
3. 다운로드한 ZIP 파일의 압축을 원하는 폴더에 풉니다.

---

## 2단계: 크롬 브라우저에 익스텐션 등록하기

1. 크롬(Chrome) 브라우저를 열고 주소창에 다음 주소를 입력하여 이동합니다.
   > **`chrome://extensions/`** (또는 크롬 우측 상단 점 3개 메뉴 -> `확장 프로그램` -> `확장 프로그램 관리`)
2. 우측 상단에 있는 **`개발자 모드` (Developer mode)** 토글 스위치를 **켬(On)**으로 활성화합니다.
3. 좌측 상단에 새롭게 나타난 **`압축해제된 확장 프로그램을 로드합니다.` (Load unpacked)** 버튼을 클릭합니다.
4. 1단계에서 압축을 풀거나 클론한 **`42gs-subjects-dashboard` 폴더(manifest.json 파일이 있는 루트 폴더)**를 선택하여 올립니다.
5. 이제 크롬 주소창 옆 확장 프로그램 목록에 **🔷 42 Subject Dashboard**가 추가됩니다.

---

## 3단계: 42 API 연동 자격 증명(UID, SECRET) 등록하기

익스텐션은 42 API와 직접 통신하여 실시간 데이터를 수집하기 때문에 본인의 API 키 정보가 필요합니다.

1. **[42 Intra Applications(https://profile.intra.42.fr/oauth/applications)](https://profile.intra.42.fr/oauth/applications)**에 접속하여 로그인합니다.
2. **`New Application`**을 누르고 아래와 같이 임의의 앱을 생성하여 **UID**와 **SECRET** 자격증명을 발급받습니다.
   - **Name**: `42 Subject Dashboard` (원하는 이름)
   - **Redirect URI(s)**: `https://github.com/tgkim3245/42gs-subjects-dashboard` (실제 쓰이지 않으므로 임의 주소 입력)
   - 자세한 앱 생성 방법은 프로젝트 폴더 내 [api_token_guide.md](docs/api_token_guide.md) 문서를 참고하세요.
3. 크롬 익스텐션 아이콘을 클릭하여 대시보드 화면을 엽니다.
4. 우측 상단의 **톱니바퀴(⚙️) 설정 버튼**을 클릭하여 설정 패널을 엽니다.
5. 발급받은 **UID**와 **SECRET** 값을 세로로 배열된 입력창에 각각 복사-붙여넣기 한 후 **`토큰 발급`** 버튼을 누릅니다.
   - `✅ 연동 완료 (토큰 보유)` 메시지가 뜨면 통신 준비가 끝납니다.

---

## 4단계: 첫 데이터 수집 및 실행

1. 설정 패널 내에 있는 **`데이터 수집 범위`**를 정합니다.
   - **전체 수집**: 모든 본과정 카뎃을 수집합니다.
   - **활동 카뎃만 수집**: 6개월 이상 블랙홀 상태인 비활동 카뎃을 제외하고 갱신하므로 수집 속도가 훨씬 빠릅니다.
2. 설정 패널 하단(또는 헤더 우측)에 있는 **`데이터 초기화 + 재수집(🔄)`** 아이콘 버튼을 클릭합니다.
3. 이제 대시보드 최상단 타이틀 옆에 **`🔄 수집 중 (대기열: N개)`** 상태가 활성화되며 실시간으로 데이터가 파싱되어 로컬 스토리지에 저장되기 시작합니다.
4. 아직 데이터를 불러오지 못한 카뎃들의 이름 옆에는 회전하는 스피너(`🔄`)가 표시되며, 백그라운드 수집이 완료되는 즉시 표에 프로젝트 진행률이 순차적으로 업데이트됩니다.
5. 대시보드 타이틀 우측에 **`✅ 모든 데이터 최신화 완료`** 문구가 뜨면 모든 수집이 끝납니다!

---

## 💡 유용한 팁 및 안내 사항
* **데이터 보존**: 한 번 수집이 완료되면 설정에서 강제로 재수집(🔄)을 누르거나 캐시를 비우지 않는 한, 브라우저가 꺼져도 로컬 데이터가 보존되어 켜자마자 즉시 로딩됩니다.
* **관심 카뎃(별표)**: 이름 옆의 별표(☆)를 누르면 관심 카뎃으로 등록할 수 있으며, 헤더의 `⭐ 관심 카뎃` 탭을 이용해 해당 인원들만 빠르게 필터링할 수 있습니다.
